# 📦 E-Commerce Orders & Delivery Tracker Dashboard

A Power BI dashboard designed to monitor and analyze e-commerce order and delivery performance. This project helps the operations team track delivery efficiency, identify delays, and make data-driven decisions using interactive visualizations and KPIs.

---

## 📌 Project Overview

The goal of this project is to provide a real-time view of order and delivery performance using Power BI. It enables business users to monitor key metrics such as on-time delivery percentage, average delivery time, daily order trends, and category-wise performance.

---

## 🎯 Business Problem

The operations team was unable to efficiently monitor:

- On-Time Delivery (OTD) Rate
- Average Delivery Days
- Daily Order Trends
- Product Category Performance
- Order-Level Delivery Details

Without a dashboard, decision-making was slow because data had to be analyzed manually.

---

## 📊 Dashboard Features

- ✅ KPI Cards
  - Total Orders

  - Average Delivery Days

- 📈 Daily Order Trend

- 📊 Category-wise Order Analysis
